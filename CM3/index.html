Les répétitions

Jeu de Nim

Problème :
Il faut pouvoir faire un tour de jeu jusqu'à ce que le nombre de bâtonnets total arrive à $0$. Il faut donc répéter des tours de jeu jusqu'à victoire d'un des joueurs.

Une définition générale

En programmation informatique, une répétition ou boucle est une séquence d'instructions qui se répète jusqu'à ce qu'une certaine condition soit atteinte
  • Une suite d'opérations est effectuée : i.e. création de variable ou modification de valeur..
  • Une condition est vérifiée à la fin de la suite d'opérations : i.e. un compteur a atteint un nombre prescrit
  • Si la condition est vérifiée, on refait la suite d'opérations, sinon on passe à la suite

Un exemple : répéter la même tâche encore et encore

Imprimer les chiffres de 1 à 5


print(1)
print(2)
print(3)
print(4)
print(5)

Pourquoi écrire la même fonction 5 fois ??

Ça a l'air mieux


for i in range(1, 6):
    print(i)

Il y a deux grandes classes de répétitions

  • Les répétitions où le nombre de fois est connu en avance ➡️ for
  • Les répétitions dont le nombre n’est pas connu a priori ➡️ while

L'instruction For

Pour un nombre connu d'itérations :

On sait à l'avance combien de fois il faut répéter une séquence d'opérations.
Par exemple :
  • Calcul des $n$ premiers termes d'une suite
  • Jeux avec un nombre de tours définis

Instruction for

Dans la plupart des langages, une boucle for définit la variable sur laquelle boucler et sa valeur initiale, jusqu'à ce que la condition soit remplie, et comment changer la condition à chaque itération


for int i = 1; i <= 5; i++ {
	...
}

Python est différent

Le for est défini sur une liste d'éléments itérables. A chaque étape, la variable prend la valeur de l'itération en cours


for var in iterable:
	statement(s)

Remarquer l'indentation...

Un itérable, c'est un objet qui permet de produire une séquence..

  • listes, tupes, dictionnaires (cf CM5)
  • iterable range, enumerate, etc

Un exemple en pratique


for i in range(1, 6):
    # A chaque étape i a une valeur comprise entre 1 et 5
    ...

Fonction range: Return an object that produces a sequence of integers from start (inclusive) to stop (exclusive) by step. range(i, j) produces i, i+1, i+2, ..., j-1.

def moyenne(liste):
	resultat = 0
	for x in liste:

	resultat = resultat / len(liste)

	return resultat

La fonction enumerate

Permet de récupérer à la fois l’indice et le contenu des éléments d’un objet itérable


def moyennePonderee(liste, poids):
	numerateur = 0
	denominateur = 0

	for i,x in enumerate(liste):
		numerateur = numerateur + x * poids[i]
		denominateur = denominateur + poids[i]
	return numerateur / denominateur

La fonction zip

Permet d’agréger plusiers éléments itérables


def moyennePonderee(liste, poids):
	numerateur = 0
	denominateur = 0

	for x, p in zip(liste, poids):
		numerateur = numerateur + x * p
		denominateur = denominateur + p
	return numerateur / denominateur

L'instruction While

Pour une nombre Indéfini de répétitions

Quand on ne sait pas à l'avence combien de fois il faudra répéter une séquence d'instructions.

Exemples

  • Demander à l'utilisateur un nombre respectant une condition..
  • Déplacer un personnage à l'écran jusqu'à ce qu'il arrive à une destination donnéee

Instruction while

Permet de répéter une séquence d’instructions tant qu’une condition est vraie


i = 1
while i <= 5:
	print(i)
	i += 1

C'est fondamentalement comme un if mais répète le code interne jusqu'à ce que la condition soit fausse

Il convient de toujours vérifier que la séquence d’instructions fait évoluer la condition de sorte à ce qu’elle devienne fausse à un moment !

Essayez plutôt ceci


i = 1
while i <= 5:
	print(i)
